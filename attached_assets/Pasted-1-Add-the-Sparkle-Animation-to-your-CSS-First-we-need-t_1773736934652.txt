1. Add the "Sparkle" Animation to your CSS
First, we need to define the movement. Open your src/index.css or globals.css in Replit/Cursor and add this at the bottom:

CSS
@keyframes heartbeat {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.2); }
}

@keyframes sparkle {
  0%, 100% { opacity: 0; transform: scale(0) rotate(0deg); }
  50% { opacity: 1; transform: scale(1.2) rotate(90deg); }
}

.animate-heartbeat {
  animation: heartbeat 1.5s ease-in-out infinite;
}

.animate-sparkle {
  animation: sparkle 2s ease-in-out infinite;
}

TypeScript

3. How to make it work
Paste the CSS into your CSS file so the animate-heartbeat and animate-sparkle classes exist.

Paste the JSX into your Dashboard.tsx where you want the greeting to show.

Check the "Vibe": Because we used backdrop-blur-xl, the sparkles will look like they are floating behind a piece of frosted glass—very similar to high-end Apple or Google designs.
